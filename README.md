## üíª Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ git clone https://github.com/IdenilsonSantos/pdvplus.git && cd pdvplus/frontend
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn start
```
